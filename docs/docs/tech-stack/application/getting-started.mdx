# 시작하기

:::tip

이 문서에서는 애플리케이션 개발환경 구축과 디버깅 애플리케이션 설치에 관한 내용을 다룹니다. 데모 애플리케이션을 바로 체험해보고 싶으시다면 [이 문서](/docs/demo)를 참고하세요.

:::

:::info

LOCA 애플리케이션은 React Native를 기반으로 하여 제작되었습니다. 관련 개발환경 구축을 위해서는 [React Native 공식 문서](https://reactnative.dev/docs/environment-setup)를 참고하세요.

:::

## 1. 프로젝트 설치

```bash
git clone https://github.com/osamhack2021/web_app_LOCA_DimiTiger.git
```

## 2. 패키지 설치

```bash
cd APP
yarn install
```

### 2-1. iOS cocoapods 라이브러리 설치 (macOS에서만 해당)

```bash
cd ios
pod install
```

## 3. 애플리케이션 실행

[React Native 공식 문서](https://reactnative.dev/docs/running-on-device) 참고

:::info

LOCA 애플리케이션은 실제 기기에서만 동작하는 기능(NFC, 블루투스 비콘 감지)이 포함되어 있습니다. 시뮬레이터에서 실행하여도 애플리케이션의 UI를 체험해볼 수 있으나, 전체 기능의 동작을 확인하기 위해서는 실제 기기에 앱을 설치하여 실행하는 것을 권장합니다.

:::
