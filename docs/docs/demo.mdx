# 데모 서비스 체험하기

서비스의 주요 기능들을 쉽고 빠르게 체험해보실 수 있도록 데모 서비스를 운영 중입니다.

## 🖥 관리자 웹 대쉬보드

웹으로 제공되는 관리자 대쉬보드는 [이곳](https://loca.kimjisub.me)에서 만나보실 수 있습니다.

## 📱 사용자 모바일 애플리케이션

사용 중인 모바일 기기에 데모 애플리케이션을 설치하여 화려한 UI 애니메이션과 함께 제공되는 LOCA의 강력한 애플리케이션을 경험해보실 수 있습니다.

### 🤖 Android 애플리케이션

Android 애플리케이션은 Google Play 스토어 내부 앱 공유를 통해 제공됩니다. [링크](https://play.google.com/apps/test/RQY7HmDqfQY/ahAIGJVrcouBoUGeodKHVl5SS3mjIhtsN1wr8yRwoXXq1zQ7bg5W91lCCxm2l53qnewU_4M08llUs-O_LZgDWYzeMW)

:::tip

Google Play 내부 앱 공유 사용을 위해서는 Google Play 애플리케이션 내에서 사전 설정이 필요합니다. 이와 관련해서는 다음 링크들을 참고하세요.

[Google 공식 문서](https://support.google.com/googleplay/android-developer/answer/9844679#on)

[관련 블로그 포스트](https://devuryu.tistory.com/333)

:::
