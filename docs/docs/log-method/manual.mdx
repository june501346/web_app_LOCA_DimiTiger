# 수동 변경

NFC나 비콘을 적용하기 어려운 위치가 있을 수 있습니다. 이 경우에는 사용자가 직접 위치 목록에서 선택하여 변경된 위치를 보고할 수 있습니다.

## 장점

- 별도로 NFC나 비콘을 배치하지 않아도 돼 설치 비용 등이 절감
- 앱이 NFC나 비콘을 탐지할 때 까지 기다릴 필요 없이 사용자가 즉시 적용 가능

## 단점

- 임의로 위치를 변경 가능하게 하기 때문에 사용자의 자율성에 의존해야 함
