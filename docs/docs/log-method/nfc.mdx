# NFC

**NFC**는 Near Field Communication의 약자로, 근거리에서 접촉을 통해 이루어지는 무선 통신 기술입니다.

각 장소에 NFC태그를 비치한 후 사용자의 모바일 기기를 접촉하면 LOCA 모바일 앱이 실행되며, 접촉된 NFC 태그에 해당하는 장소로의 변경화면으로 자동 이동됩니다.

:::tip

Android 기기에서는 NFC태그에 접촉되는 즉시 앱이 실행되지만, iOS 기기에서는 운영체제 정책 상 NFC태그 접촉시 발생하는 알림을 클릭해야 앱이 실행됩니다.

:::

## 구동 원리

장소에 비치할 NFC 태그에 장소의 id가 포함된 딥링크 URL을 Write합니다. 이후 해당 NFC 태그에 기기를 태그하면 앱은 링크에서 장소의 id를 읽어와 해당 장소의 변경 화면으로 이동합니다.

딥링크 URL형식은 다음과 같습니다.

`https://api.loca.kimjisub.me/link/location-log/:locationId`

## 👍 장점

- NFC 태그에 사용자의 기기를 태그하여 신속한 위치 보고 가능
- 비콘에 비해 설치 비용이 저렴

## 👎 단점

- 사용자가 NFC 태그가 있는 물리적인 위치까지 이동하여 직접 기기를 태그해야 작동 가능
