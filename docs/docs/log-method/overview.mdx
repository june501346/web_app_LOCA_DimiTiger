# 개요

LOCA 서비스는 병력의 위치 보고를 위해 세 가지의 방식을 제공합니다.

다양한 방식의 제공을 통해 부대 여건에 맞추어 유동병력관리체계의 구축이 가능하도록 합니다.

## NFC

**NFC**는 Near Field Communication의 약자로, 근거리에서 접촉을 통해 이루어지는 무선 통신 기술입니다.

각 장소에 NFC태그를 비치한 후 사용자의 모바일 기기를 접촉하면 LOCA 모바일 앱이 실행되며, 접촉된 NFC 태그에 해당하는 위치로의 변경화면으로 자동 이동됩니다.

[NFC 자세히 보기](nfc)

## Beacon

**비콘(Beacon)**은 블루투스 4.0 기반의 프로토콜을 이용해 주변 기기들에게 신호를 전파하는 장치입니다.

각 장소에 비콘을 비치해 놓은 후, 사용자가 모바일 기기를 소지하고 해당 장소에 진입하면 위치에 진입했다는 푸쉬 알림이 표시되며, 해당 알림을 선택하여 앱이 실행되면 해당되는 위치로의 변경화면으로 자동 이동됩니다.

[비콘 자세히 보기](beacon)

## Manual(수동)

NFC나 Beacon을 이용하기에 모두 부적합한 경우가 있을 수 있습니다. 이 경우에는 사용자가 애플리케이션을 직접 실행하여 등록된 위치 목록중에서 선택하여 위치를 변경할 수 있습니다.

[수동 변경 자세히 보기](manual)
