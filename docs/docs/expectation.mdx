# 기대효과

### 👥 사용자 기대효과

- NFC와 비콘을 활용한 쉽고 간편한 위치 보고
- 긴급 신고 기능을 통해 정확하고 신속한 사고 보고
- 공지사항 기능을 통해 언제 어디서든 전파 내용 확인

### 🖥 관리자 기대효과

- 관리자 대시보드를 통해 병력 위치를 한 눈에 파악
- 사고 발생 또는 감염병 확진자 발생 시 관련 인원에 대한 정확한 동선 파악

### 🌈 다채로운 활용 방안

- 코로나19 상황에서의 활용: 확진자 발생 시 확진자의 이동 동선과 접촉자를 즉각적으로 확보할 수 있습니다.
- 기반 마련을 통한 무한한 확장: NFC와 비콘과 연동된 시스템이 구축된다면 보다 넓은 분야에 활용할 수 있습니다. 그 예로, 식당의 식수 인원 확인에 NFC 태그 방식을 활용한다면 쉽고 빠르게 식사 현황을 파악할 수 있습니다.
