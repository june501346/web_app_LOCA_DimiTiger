# 개요

LOCA 서비스의 관리자는 웹을 통해 관리자 서비스에 접근합니다.

웹 대쉬보드를 통해, 각 부대 행정반의 업무부터 유동병력 현황까지 한 눈에 볼 수 있게 도와줍니다.
