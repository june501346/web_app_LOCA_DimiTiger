# 2. 한 눈에 보는 유동병력 현황

관리자는 웹 대쉬보드를 통해, 현재 병력의 위치, 인원 정보, 장소별 병력 현황을 손쉽게 파악할 수 있습니다.

### 커스터마이징 가능한 유동병력 현황판

- 사용자의 니즈에 맞게, Circle Packing(원)과 Tree Map(사각형) 중 디자인을 선택
- Circle Packing : **전체적인** 병력현황 파악 용이
- Tree Map : 장소 내 병력의 **세부정보** 파악 용이
