# 3. 위치 기반 긴급보고 체계

앱 내 긴급신고 기능을 이용하여 유사시 사용자가 신속히 관리자에게 보고할 수 있습니다.

### 신속한 보고 체계

- 본인이 또는 인접 전우를 통하여 보고하는 현행의 시간적 지연을 해결
- 실시간 데이터 처리 방식으로 보고 즉시 관리자 인지 가능
- 추가로 보고되는 사항을 실시간으로 업데이트하여 확인 가능

### 손쉬운 보고 내용 파악

- 관리자는 신고된 사항을 인원별, 시간대별로 정제된 정보로 확인
- 보고된 사항과 최근 위치 기록을 하눈에 확인할 수 있어 상황 파악에 용이

### 보고 정보 데이터베이스화

- 과거 보고 데이터 조회를 위해, 조치 완료된 신고 내용도 기록 후 제공
