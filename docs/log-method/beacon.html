비콘(Beacon)

비콘(Beacon)은 블루투스 4.0 기반의 프로토콜을 이용해 주변 기기들에게 신호를 전파하는 장치입니다.

사용자가 비콘이 비치된 위치에 기기를 소지하고 진입하면 위치에 진입했다는 푸쉬 알림이 표시되며, 해당 알림을 선택하여 앱이 실행되면 해당되는 위치로의 변경화면으로 자동 이동됩니다.

애플리케이션을 실행 중인 경우 비콘을 이용해 탐지된 위치들의 목록과 각각의 대략적인 거리를 확인할 수 있습니다.

tip

앱이 실행되어 있지 않은 백그라운드 상태에서, iOS 기기에서는 운영체제의 스케줄에 따라 비콘의 신호를 수시로 확인하여 위치에 진입한 시간과 거의 동시에 알림이 표시되지만, Android 기기에서는 운영체제 정책 상 배터리를 절약하기 위해 비콘의 스캔 주기가 제한되어 있습니다. LOCA의 경우에는 Android 기기에서 최대 5분 내에 변경된 위치를 탐색할 수 있도록 설정해두었습니다.

구동 원리#

애플리케이션이 실행되면 비콘 목록을 API에서 불러온 후 해당 비콘들에 대한 검색을 시작합니다. 검색을 설정한 비콘이 탐지되었을 때, 저장되어 있는 비콘-장소 매핑 목록에서 장소의 id를 찾아 해당 장소의 변경화면으로 이동됩니다.

LOCA는 비콘 규격으로 Apple iBeacon을 사용합니다. iBeacon은 UUID와 각각 2Byte의 Major, Minor 값으로 이루어진 신호를 Broadcast합니다. 비콘을 LOCA 서비스에서 운영하기 위해서는 관리자 웹페이지의 비콘 관리 페이지에서 각각의 비콘에 설정된 UUID, Major, Minor 값을 등록하고 장소를 할당해 주어야 합니다.

비콘 추가 페이지

👍 장점#

  • 사용자가 보고된 위치와 다른 위치에 있는 것으로 확인되었을 경우 기기에 알림을 표시하여 신속한 위치 보고 유도
  • NFC와 달리 비콘의 신호가 도달하는 범위 내에서라면 기기의 직접적인 접촉 없이 손쉽게 현재 위치 확인 후 보고 가능

👎 단점#

  • NFC 태그에 비해 비교적 가격대가 높은 비콘을 구매해 설치해야하므로 비용적 부담이 있음
